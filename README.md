# node-access-refresh-token
node-access-refresh-token

> cookie-parser, jsonwebtoken 이용

### Access Token & Refresh Token

 목록 | Access Token | Refresh Token
---|---|---|
저장소 | Client header Cookie | 세션, mySQL, DB
만료시간 | 보통 Refresh Token보다 짧음 | 보통 Access Token보다 김
